HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element gives the webpage information about the data
  • The section element allows us to organise the page into sections.
  • you can create headings using h1, h2, h3, etc. elements.
  • HTML uses the id attribute to identify elements and select them.
  • The ul element creates an unordered list.
  • The li element can be used to create a bullet pointed list.
